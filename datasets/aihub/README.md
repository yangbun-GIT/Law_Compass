# AI Hub 로컬 데이터 작업 폴더

이 폴더는 AI Hub 교통사고/차량 인지 데이터셋을 로컬에서 확인하기 위한 전용 위치입니다.

실제 데이터 파일, 압축 파일, 영상, 라벨 원본, 모델 가중치는 Git에 올리지 않습니다. 아래 하위 폴더에 넣으면 `.gitignore` 규칙에 따라 제외됩니다.

## 폴더 용도

| 폴더 | 용도 |
| --- | --- |
| `samples/` | AI Hub의 `샘플(경량) 데이터`를 먼저 넣는 위치 |
| `raw/` | 승인 후 받은 원본 대용량 데이터 또는 분할 압축 해제 결과 |
| `processed/` | 프로젝트 입력 계약에 맞게 변환한 라벨/프레임/메타데이터 |
| `models/` | 로컬 YOLO 등 비전 모델 가중치 또는 실험 산출물 |

## 우선 넣을 것

1. AI Hub 페이지에서 받은 `샘플(경량) 데이터`를 `samples/`에 넣습니다.
2. 압축을 풀면 원본 영상/이미지와 라벨 JSON 구조가 보이도록 유지합니다.
3. 전체 데이터는 용량이 크므로 샘플 구조 확인 전에는 `raw/`에 넣지 않습니다.

## 주의

- AI Hub 이용약관과 승인 조건을 지켜야 합니다.
- 개인정보 또는 식별 가능 정보가 포함된 파일은 Git, 문서, 로그에 포함하지 않습니다.
- 모델 학습/평가에 필요한 최소 subset만 `processed/`로 변환합니다.
