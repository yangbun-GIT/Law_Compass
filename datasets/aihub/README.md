# AI Hub 로컬 데이터 작업 폴더

이 폴더는 AI Hub 교통사고/차량 인지 데이터셋을 로컬에서 확인하기 위한 전용 위치입니다.

원본 전체 데이터는 테라 단위라 로컬에 받지 않는 것을 원칙으로 합니다. 실제 데이터 파일, 압축 파일, 영상, 라벨 원본, 모델 가중치, AI Hub Shell 다운로드 결과는 Git에 올리지 않습니다.

## 데이터셋별 폴더

| 폴더 | 용도 |
| --- | --- |
| `traffic-accident-video/` | AI Hub `교통사고 영상 데이터` 작업 위치 |
| `vehicle-person-recognition/` | AI Hub `차량 및 사람 인지 영상` 작업 위치 |

각 데이터셋 폴더는 `samples/`와 `aihubshell/`만 둡니다.

## 우선 넣을 것

1. AI Hub 페이지에서 받은 `샘플(경량) 데이터`는 해당 데이터셋의 `samples/`에 넣습니다.
2. AI Hub Shell의 파일 목록, 선택 다운로드 결과, shell 실행 로그는 해당 데이터셋의 `aihubshell/`에 넣습니다.
3. 전체 데이터셋 다운로드는 하지 않습니다. Shell을 쓰더라도 먼저 파일 목록에서 `filekey`를 확인한 뒤 필요한 작은 파일만 선택합니다.

## 주의

- AI Hub 이용약관과 승인 조건을 지켜야 합니다.
- 개인정보 또는 식별 가능 정보가 포함된 파일은 Git, 문서, 로그에 포함하지 않습니다.
- API key는 `.env`나 개인 보관 위치에만 두고, 이 폴더나 문서에 넣지 않습니다.
